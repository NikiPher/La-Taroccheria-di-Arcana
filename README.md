# 🔮 La Taroccheria di Arcana

Un'applicazione web interattiva per esplorare il mondo affascinante dei Tarocchi. Scopri i significati delle carte, ottieni letture personalizzate e approfondisci la saggezza arcana.

## ✨ Caratteristiche

- 🃏 **Database completo dei Tarocchi** - Tutte le carte dei Tarocchi con descrizioni dettagliate
- 🎲 **Letture casuali** - Ottieni letture personalizzate e consigli
- 🎨 **Interfaccia elegante** - Design moderno e intuitivo
- 📱 **Responsive** - Perfetto su desktop, tablet e mobile
- 🌙 **Tema scuro/chiaro** - Supporto per entrambi i temi
- 💾 **Salva le tue letture** - Mantieni la cronologia delle tue consultazioni

## 🚀 Avvio rapido

### Prerequisiti
- Node.js 16+ 
- npm o yarn

### Installazione

```bash
# Clona il repository
git clone https://github.com/NikiPher/La-Taroccheria-di-Arcana.git
cd La-Taroccheria-di-Arcana

# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev
```

L'app sarà disponibile su `http://localhost:5173`

### Build per la produzione

```bash
npm run build
```

## 📦 Stack Tecnologico

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Form Management**: React Hook Form + Zod
- **Visualizzazioni**: Recharts, Three.js
- **Router**: React Router v6
- **State Management**: TanStack Query
- **Animazioni**: Framer Motion

## 📜 Script disponibili

- `npm run dev` - Avvia il server di sviluppo
- `npm run build` - Crea una build ottimizzata per la produzione
- `npm run preview` - Anteprima della build di produzione
- `npm run lint` - Esegui il linter
- `npm run lint:fix` - Correggi i problemi del linter
- `npm run typecheck` - Controlla i tipi TypeScript
